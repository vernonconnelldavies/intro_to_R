For my module Introduction to R assignment I had to create a basic Shiny app that functions.
I chose to display a (fictitious) high school student class's scores for different subjects. The user then has the option of downloading a basic PDF report that has some basic info on the class results.

The Shiny app itself has been uploaded to https://vernonconnelldavies.shinyapps.io/intr0_to_r/
